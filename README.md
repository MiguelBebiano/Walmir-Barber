# Walmir Barbearia

Este projeto é um site/aplicação para a barbearia **Walmir Barber**, desenvolvido em **Front-end** com foco em apresentar serviços, informações de contato e agendamento.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Node.js
- MySQL (para gerenciamento de dados)

## 📂 Estrutura do projeto
- `index.html` → página inicial
- `css/` → estilos
- `db/` → configuração da conexão com banco de dados
- src/
├── `config/`
│    └── `database.js`   # Configuração da conexão com o banco
├── `controllers/`       # Lógica de controle das rotas
├── `models/`            # Definição das entidades e consultas ao banco
├── `routes/`            # Definição das rotas da aplicação
├── `services/`          # Regras de negócio e serviços auxiliares
└── `app.js`             # Ponto de entrada da aplicação
