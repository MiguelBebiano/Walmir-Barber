# Walmir Barbearia

Este projeto é um site/aplicação para a barbearia **Walmir Barber**, desenvolvido em **Front-end** com foco em apresentar serviços, informações de contato e agendamento.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Node.js
- MySQL (para gerenciamento de dados)

## 📂 Estrutura do projeto
- `index.html` → página inicial
- `css/` → estilos
- `js/` → scripts
- `db/` → configuração da conexão com banco de dados
